## Deploy to Vercel

1. Go to [vercel.com/new](https://vercel.com/new)
2. Import this GitHub repo: `0x0dabid/cognoxide`
3. Framework Preset: **Other**
4. Click Deploy
5. Add custom domain: `cognoxide.com`
